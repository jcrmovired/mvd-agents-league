# ReportMind

Este proyecto consiste en el desarrollo de un agente para la competencia **Microsoft Agent Battle**. El objetivo principal del agente es asistir a los usuarios en la interpretación y análisis de documentos relacionados con el área de reporting.

## Propósito

El agente está diseñado para:
- Analizar documentos de reporting.
- Ayudar a los usuarios a comprender y extraer información relevante de reportes.
- Facilitar la toma de decisiones basada en datos presentados en los documentos.
- **Generar visualizaciones y gráficas** de los datos analizados usando matplotlib.

## Nuevas Capacidades - MCP Matplotlib

Se ha integrado un servidor MCP (Model Context Protocol) para matplotlib que permite:
- Crear gráficas de barras, líneas, scatter, histogramas y pie charts
- Generar visualizaciones automáticas de datos de reportes
- Exportar gráficas en formato PNG de alta calidad
- Integración directa con el análisis de documentos

### Tipos de gráficas soportadas:
- **Barras**: Para comparaciones categóricas
- **Líneas**: Para tendencias temporales
- **Scatter**: Para correlaciones
- **Histogramas**: Para distribuciones
- **Pie**: Para proporciones

## Estado del proyecto

Actualmente, el desarrollo se encuentra en curso y se están implementando las capacidades principales del agente, incluyendo la nueva funcionalidad de visualización de datos.
